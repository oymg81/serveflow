# ServeFlow

## Description
ServeFlow is a full-stack web application designed to help churches organize volunteer scheduling for weekly services and events. The app allows volunteers to create accounts, be assigned to a specific ministry by an administrator, and select the service times they are available to serve. Administrators can then review those selections and manage final assignments and confirmation statuses.

## Purpose
The purpose of ServeFlow is to simplify and centralize volunteer scheduling. Many churches currently manage schedules through messaging apps, spreadsheets, or verbal coordination, which often leads to confusion and miscommunication. ServeFlow provides a structured and efficient system for managing events, volunteers, and service assignments in one place.

## Inspiration
This app is inspired by real scheduling needs within church environments, where volunteers serve in different ministries and roles across multiple service times (e.g., English and Spanish services). Instead of relying on scattered communication, ServeFlow offers a clear workflow where volunteers can indicate availability and coordinators can organize and confirm schedules effectively.

## Core Features
1. Account creation for volunteers to join the system.
2. Admin assignment of users to a specific ministry (e.g., Greeters, Kids, Media).
3. Admin creation and management of service events or time slots.
4. Volunteer selection of available service times.
5. Admin confirmation of role assignments.
6. Real-time viewing of event details and assigned volunteers.

## Custom Features
- **Filtering & Sorting**: Filter volunteers or assignments by ministry or service time. Helps coordinators quickly review schedules.
- **Validation Logic**: Prevent invalid scheduling actions (e.g., past dates or duplicate assignments). Ensures data integrity and better user experience.
