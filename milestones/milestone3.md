# Milestone 3 Reflection

## 1. What did you accomplish during this milestone?
During this milestone, I pitched my project idea and set up the GitHub repository for development. I created issues based on the feature list, organized them into a project board, and defined milestones to track progress. I also began the initial setup for the application, preparing the structure for future development.

## 2. How did your project idea or plan evolve this week?
This week, my project evolved from a conceptual idea into a structured development plan. I translated features into actionable tasks using GitHub issues and organized them into a clear workflow. I also refined which features should be prioritized first, focusing on core functionality like event creation and volunteer management.

## 3. What challenges did you face during this milestone?
One of the main challenges was breaking down the project into smaller, manageable tasks that could be tracked effectively in GitHub. Another challenge was organizing the workflow in a way that aligns with future development while keeping things simple and clear.

## 4. How did your team collaborate during this milestone?
During this milestone, collaboration was limited, but I ensured progress by organizing the repository, defining tasks, and setting up the development workflow independently. The use of GitHub tools like issues and project boards helped maintain structure and clarity.

## 5. What are your next steps for the project?
The next steps are to begin full development of the application. This includes building the backend, connecting the database, implementing the main features such as user registration and event management, and continuing to track progress using the GitHub board.
