# User Stories

## Roles
1. **Admin / Coordinator**: Assign users to ministries, create and manage service events, review volunteer availability, confirm or update assignments.
2. **Volunteer**: Create an account, view assigned ministry, select available service times, view assignment status.

## User Stories
1. As a volunteer, I want to create an account so I can join the scheduling system.
2. As an admin, I want to assign a user to a ministry so they only see relevant service opportunities.
3. As an admin, I want to create service events so I can organize weekly schedules.
4. As a volunteer, I want to view available service times so I know when I can serve.
5. As a volunteer, I want to select a service time so I can indicate my availability.
6. As an admin, I want to review volunteer selections so I can finalize the schedule.
7. As an admin, I want to confirm or update assignment status so I know who is serving.
8. As a volunteer, I want to view my assigned role so I can prepare accordingly.
9. As an admin, I want to filter assignments by ministry or time so I can manage schedules efficiently.
10. As an admin, I want the system to prevent invalid data so scheduling errors are reduced.
